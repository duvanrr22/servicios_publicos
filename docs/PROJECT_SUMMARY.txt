CÓMO INSTALAR
=============

1. EN FRAPPE BENCH:

   cd ~/frappe-bench
   bench get-app servicios_publicos /ruta/a/servicios_publicos
   bench install-app servicios_publicos
   bench migrate
   bench restart

2. O USANDO GIT:

   cd ~/frappe-bench/apps
   git clone <URL_repositorio> servicios_publicos
   cd ..
   bench install-app servicios_publicos
   bench restart

3. VERIFICAR INSTALACIÓN:

   bench list-apps | grep servicios_publicos
   
   Debe aparecer: servicios_publicos
